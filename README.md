 Weather Dataset Analysis Project
📌 Project Overview
This project focuses on analyzing an Indian climate dataset (2024–2025) using Python. The dataset contains important weather parameters such as temperature, humidity, and rainfall across different cities and states.

The goal of this project is to extract, process, and understand climate data using basic data analysis techniques.

🛠️ Technologies Used
Python 🐍
Pandas 📊
NumPy 🔢
Google colab 📒
📂 Dataset Details
The dataset includes the following columns:

Date
City
State
Temperature_Max (°C)
Temperature_Min (°C)
Temperature_Avg (°C)
Humidity (%)
Rainfall (mm)
⚙️ Project Workflow
Import required libraries (Pandas, NumPy)
Load the dataset using Pandas
Extract individual columns
Convert data into NumPy arrays
Display and analyze values
💡 Key Features
Easy dataset loading using Pandas
Conversion of data into NumPy arrays
Separate handling of each weather parameter
Clean and simple data processing approach
▶️ How to Run
Install required libraries: pip install pandas numpy

Open the notebook: jupyter notebook

Run all cells step by step

📸 Output
The program prints values of:

Dates
Cities & States
Temperature (Max, Min, Avg)
Humidity
Rainfall
